## How to run
1. when start run, code makes appear 4 buttons: "List","Add schedule","Remove schedule", and "close"
2. Please click the one you want to do,
   List; show the To-Do list, what plans already keep (at first I beleive there does not have any plan wirtten inside) 
   Add/Remove schedule; you could add/remove a plan to To-Do list as write information into what code ask
   Close; Close that code and keep what plans inside To-Do list.
